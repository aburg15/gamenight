## Installation + Setup

   - Clone this repo down to your local machine: [here](https://github.com/ccollums/gamenight)
   - On the command line, type: **$ npm install**
   - On the command line, type: **$ npm start**
   - Visit http://localhost:3000/#/ in the browser.
